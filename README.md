# Mathplus-CLI-Calculator

An interactive CLI calculator supporting arithmetic operations and special extended features like exact age calculator, equation solving (upcoming).

## Features

- Basic Arithmetic Operations: Addition, Subtraction, Multiplication, Division
- Advanced Operations: Average, Power, Factorial, Prime Check, Leap Year Check
- Extended Features: Exact Age Calculator, Quadratic Equation Solver (upcoming)

## Setup Instructions

1. Clone the repository:

   ```bash
   git clone
2. Navigate to the project directory:

   ```bash
   cd Mathplus-CLI-Calculator
   ```

3. Comile the program using a C++ compiler:

   ```bash

   g++ src/*.c -o mathplus_calculator main.cpp

    ```

4. Run the program:

   ```bash

    ./mathplus_calculator

    ```
